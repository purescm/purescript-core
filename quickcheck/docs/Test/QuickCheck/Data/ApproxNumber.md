## Module Test.QuickCheck.Data.ApproxNumber

#### `ApproxNumber`

``` purescript
newtype ApproxNumber
  = ApproxNumber Number
```

A newtype for `Number` whose `Eq` instance uses an epsilon value to allow
for precision erros when comparing.
